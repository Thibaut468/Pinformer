1 - Prérequis :

	- Avoir une version de JAVA à jour pour avoir la meilleure expérience de jeu (et éviter les bugs...).

==============================

2 - Installation

Méthode 1 : Lancement via le .jar (Si ne fonctionne pas, méthode 2 plus bas)
	- Un exécutable .jar à été, pour l'occasion, mis en place pour un lancement plus simple et une portabilitée plus intéressante.
	- Double cliquer sur le fichier Pinformer.jar ou l'executer via l'invité de commande avec "java -jar Pinformer.jar"
	- Vous voilà sur l'écran principal ! Passer à la partie 3 !

Méthode 2 : Lancement du jeu par le code complet (si erreur ou bug avec le .jar)
	- Une archive ZIP nommée PinformerCode.zip contient l'intégralité des fichiers sources et des ressources nécessaires à l'exécution du jeu
	- Décompresser-là.
	- Trouver le fichier Launch.java, l'ouvrir avec votre IDE, et lancer son exécution. Normalement, l'archive contient tous les .class déjà prêts et compilés.
	- Vous voilà sur l'écran principal ! Passer à la partie 3 !
	
==============================

3 - Mode d'emploi rapide

	- Une fois sur l'écran principal, vous avez accès à 4 boutons : Jouer, Paramètres, Crédits, Quitter 
		- "Jouer" vous amène directement sur une autre fenêtre présentant vos scores sur les 4 niveaux différents ainsi que des boutons permettant d'accéder à chaque niveau. Pas de panique, uniquement les niveaux débloqués possèdent un bouton mauve, les autres sont grisées et inutilisables. Une fois le niveau N terminé, le niveau N+1 deviendra accessible.

		- "Paramètres" vous donne deux libertés en plus : choisir vos touches préférés et votre personnage ! Chacun ses goûts ! 

		- "Crédits" est une petite feature indiquant des informations sur le jeu.

		- "Quitter" éteint simplement le programme.

	- Dès lors que vos paramètres favoris seront pris, partez directement sur le bouton "Jouer" !

	- Si c'est la première fois que vous jouez, vous n'aurez accès qu'au niveau 1. Cliquer sur ce dernier, une pop-up vous informe le chargement, puis attendez patiemment.....

	- Vous voilà en jeu ! Sur votre droite, une petite pop-up vous indique, quand vous le voulez, d'appuyer sur la barre d'ESPACE autant de fois que vous le souhaitez durant 5 secondes pour atteindre les environs d'un carré rouge (indicateur) se déplaçant sur l'écran. Attention à bien visez et ne pas partir en dehors de la carte... Sinon GAME OVER !

	- Une fois votre projection sur le plateau de jeu effectué, votre objectif est d'arrivée à la plateforme verte d'arrivée ! Mais attention, votre chemin sera semé d'embuche (missiles, monstres à distances, monstres au corps à corps (petite astuce : pour les éliminer, toucher les par le haut ;) ). Quelques objets seront présents pour vous aider dans votre quête.... ou pas.

	- A cet instant, 2 possibilités : VICTOIRE ou GAME OVER.

		- "Victoire" : Félicitations, vous venez de débloquer le prochain niveau avec brio ! Quitter le jeu par la croix rouge et relancer le programme pour accéder à la suite de PINFORMER.

		-"GAME OVER" : Désolé... Ce sera pour une prochaine fois! Quitter le jeu par la croix rouge et relancer le programme pour retenter à nouveau!
